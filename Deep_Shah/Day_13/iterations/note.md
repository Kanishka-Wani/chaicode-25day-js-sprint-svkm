# Lecture 27 – While Loop & Do While Loop

## Why Multiple Loops?

JavaScript provides multiple loop types:

- for loop
- while loop
- do...while loop

All perform repetition, but their syntax and use cases differ.

---

## While Loop

### Syntax

```js
let index = 0;

while (index <= 10) {
    console.log(index);
    index++;
}
